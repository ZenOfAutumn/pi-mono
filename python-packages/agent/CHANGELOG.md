## [0.57.1] - 2025-01-15

### Added

- Initial Python port of pi-agent-core
- Core Agent class for state management
- Event streaming system with async iterators
- Tool execution framework
- Message queue management (steering and follow-up)
- Type definitions for agent, messages, tools, and events
- Agent loop implementation with LLM streaming
- Support for thinking/reasoning levels
- Session management and API key resolution

### Fixed

- Initial release
